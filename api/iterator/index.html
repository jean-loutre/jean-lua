    iterator¶

    Iterator, with map / filter functions and the like.

    Methods¶

    init()¶

    Signature

    function iterator:init(iterator: func(:any,:any?):any?, invariant: any?, control: any?) -> nil
    Parameter Type Description Default
    iterator* func(:any,:any?):any? The iterator function.
    invariant* any? Invariant state passed to the iterator.
    control* any? Initial control variable passed to the

    Notes

    Can be initialized with a statefull iterator closure, or a stateless iterator iterator. In the later case, invariant and control are the states needed by the stateless iterator. For example, you can call :

    <a href='/api/map/#iterator'><a href='/api/list/#iterator'>Iterator</a></a>(ipairs({"some", "list"}))

    call()¶

    Call all element in the iterator with the given arguments.

    Signature

    function iterator:call(...: any) -> An
    Parameter Type Description Default
    ...* any Arguments to call iterator elements with.
    Returns Description
    An iterator of the result of the calls.

    Usage

    local methods = iter({add, multiply, divide})
local operations = methods:call(4, 2):to_list()
-- { 6, 8, 2 }
